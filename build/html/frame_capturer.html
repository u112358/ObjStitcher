next_frame()[source]¶

Capture the next frame based on the current mode. In ‘local’ mode, it retrieves the next frame from a local source. In ‘camera’ mode, it raises a NotImplementedError as capturing from a camera is not yet implemented. :raises IndexError: If there are no more frames to capture in ‘local’ mode. :raises NotImplementedError: If the mode is set to ‘camera’.

Returns:

The next frame in ‘local’ mode.

Return type:

frame
